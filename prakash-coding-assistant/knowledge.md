# MERN Knowledge Base

## MongoDB
- NoSQL Document Database
- JSON-like documents store hote hain
- Mongoose ORM use karte hain

## Express.js
- Node.js ka web framework
- REST APIs banane ke liye
- Middleware concept important hai

## React.js
- Frontend UI library
- Component-based architecture
- Important Hooks:
  - useState — state manage karna
  - useEffect — side effects
  - useContext — global state
  - useRef — DOM access

## Node.js
- JavaScript runtime
- Backend server banane ke liye
- npm — package manager

## Redis & Background Workers
- Redis (Upstash): Remote cloud Redis for session/queue management.
- BullMQ: Node.js library for handling message queues.
- Worker: Separate process to handle background tasks (like notifications).

## Important Concepts
- REST API
- CRUD Operations
- JWT Authentication
- Async/Await
- Error Handling
- Environment Variables (.env)
- Debounce (Search optimization in Analytics)
- SaaS Architecture (Multi-facility management)

## Common Packages
- mongoose — MongoDB ODM
- express — Web framework
- cors — Cross origin
- dotenv — Environment variables
- bcrypt — Password hashing
- jsonwebtoken — JWT tokens
- axios — HTTP requests (frontend)
- bullmq — Queue management
- ioredis — Redis client for Node.js
